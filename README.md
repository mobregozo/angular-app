# angular-app
Simple AngularJs Application

## Install Node & npm
+ View from https://nodejs.org/download/

## Install Dependencies

+ `npm install grunt -g`

+ `npm install grunt-cli -g`

+ `npm install bower -g`

## Install dependencies UI

+ `bower install`


## Build & development

+ Run `grunt` for building and `grunt dev` for preview.
